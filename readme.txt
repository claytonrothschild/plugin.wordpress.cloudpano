The Cloudpano 360 Tour Embedder is a powerful WordPress plugin that allows you to easily embed Cloudpano tours into your WordPress website. With this plugin, you can create immersive and interactive 360-degree tours and showcase them seamlessly on your website.

== Installation ==
Activate the plugin through the \'Plugins\' menu in WordPress.
Start embedding Cloudpano tours into your posts or pages using the provided shortcode.


== Frequently Asked Questions ==
Q: Can I embed multiple tours on a single page?
A: Yes, you can embed multiple tours on a single page by using different shortcodes for each tour.
